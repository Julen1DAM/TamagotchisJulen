# TamagotchisJulen

¡Bienvenido a **TamagotchisJulen**!  
Este proyecto es un juego de Tamagotchi desarrollado en Java donde podrás cuidar de tus mascotas virtuales, alimentarles, jugar con ellas y asegurarte de que estén felices y sanas.

---

## Requisitos Previos

- **Java JDK** (versión recomendada: 8 o superior)
- IDE recomendado: **Eclipse** o **Visual Studio Code** con extensión de Java

---

## Importar el proyecto en Eclipse

1. **Clonar o descargar el repositorio**
    ```bash
    git clone https://github.com/Julen1DAM/TamagotchisJulen.git
    ```
    O descarga el ZIP desde GitHub y descomprímelo donde prefieras.

2. **Importar el proyecto en Eclipse**
    - Abre Eclipse.
    - Ve a `File > Import... > Existing Projects into Workspace`.
    - Selecciona la carpeta raíz del proyecto (`TamagotchisJulen`).
    - Haz clic en `Finish`.

3. **Compilar y ejecutar**
    - Haz clic derecho en la clase principal (por ejemplo, `Main.java`) y selecciona `Run As > Java Application`.

---

## Importar el proyecto en Visual Studio Code

1. **Clonar o descargar el repositorio**
    ```bash
    git clone https://github.com/Julen1DAM/TamagotchisJulen.git
    ```
    O descarga el ZIP y descomprímelo.

2. **Abrir la carpeta del proyecto**
    - Ve a `Archivo > Abrir carpeta...` y selecciona `TamagotchisJulen`.
   
3. **Instalar las extensiones necesarias**
    - Instala las extensiones recomendadas: `Java Extension Pack`.
   
4. **Compilar y ejecutar**
    - Abre la clase principal (probablemente `Main.java`) y pulsa el botón de ejecutar (▶️) o usa el comando `Run Java`.

## Estructura General del Proyecto

```
TamagotchisJulen/
├── src/
│   └── [paquetes y archivos fuente .java]
├── .gitignore
├── README.md
└── [demás archivos de Maven]
```
- **src/**: Contiene el código fuente. Suele dividirse en paquetes por funcionalidad (puede haber por ejemplo un paquete `model`, `controller`, `view`).
- **.gitignore**: Define los archivos y carpetas que no se deben incluir en el control de versiones.
- **README.md**: Este archivo, contiene la información básica y la guía de uso.
